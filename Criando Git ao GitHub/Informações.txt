Trabalhando Git ao gitHub
. git config --list >confirma igualdade entre repositório local e remoto
. git config --global user.email > confirma email
                    >user.name> confirma autor
. git remot add origin ( cola endereço Github) cadastra repositório
. git remot -v > lista repositórios cadastrados
. git add . > atualiza o repositório
. git commit - m (discrição do que foi atualizado) tem que informar o que se foi modificado
                                                   ou acrecentado.
. git push origin main > fundamental para reconhecimento e inclusão no repositório remoto
                           (GitHub)
